# UP_Demos

Welcome to **UP_Demos**, a collection of Advanced AI Agents and Enterprise Demonstration Projects built to showcase the power of Google Cloud AI, Vertex AI Agent Engine, and the Gemini ecosystem.

## 🚀 Featured Projects

### 📝 Document & Translation AI
* **[Translation Agent (v1)](translation/README.md)**: A standalone agent demonstrating automated layout preservation and terminology-consistent financial document translations.
* **[Enterprise Translation Pipeline (v2)](translation-v2/README.md)**: An upgraded pipeline leveraging `pdfplumber` for precise coordinate-based text extraction and multi-column visual fidelity.

### 🏦 Financial & Underwriting Agents
* **SmartUnderwriter**: Intelligent underwriting workflows with advanced document context evaluation.
* **InfiniteUnderwriter**: Next-generation continuous underwriting agent demos tailored for scale.
* **UWM (United Wholesale Mortgage)**: A suite of modular AI systems split across Tech, Non-Tech, and generic workflows tailored to enterprise banking and mortgage evaluations.

### 🏠 Domain-Specific AI
* **Open House Agent**: A conversational tool built on Google ADK that leverages live property listings and RAG to assist potential home buyers in real-time.
* **AutoNation**: An automotive-focused AI agent demo built to streamline operations and enhance retail logistics.

### ⚙️ Core Integrations
* **Vertex AI RAG Engine**: Explorations and evaluations of RAG architectures overlaid on prebuilt LangChain/Agent Engine templates.
* **MCP Integrations**: (Model Context Protocol) Integration layers connecting external platforms (like BigQuery, Snowflake, and Salesforce) seamlessly securely into Gemini workspaces.

## Getting Started
Each project folder typically comes with its own respective `README.md` or `GEMINI.md` to guide you through installation, Python virtual environment (`.venv`) setups, and Google Cloud SDK (`gcloud`) deployment instructions.

Enjoy exploring the demos!
